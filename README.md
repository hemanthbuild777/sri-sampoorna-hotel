# Sri Sampoorna Hotel Website 🍽️

A complete restaurant web application built for a real local business in Pedda Amberpet, Hyderabad.

## Features
- 🍽️ Dynamic menu with category filters and search
- 🛒 Cart with quantity management
- 💳 Payment options (UPI, Card, Cash on Delivery)
- 📦 Order tracking (Placed → Preparing → Ready)
- 📱 WhatsApp ordering integration
- 🪑 Table booking system
- 🎟️ Coupon code system
- ⭐ Customer feedback with star ratings
- 🔐 Admin dashboard (add/delete menu items, Today's Special)
- 📱 Fully responsive design

## Tech Stack
HTML5 | CSS3 | JavaScript | LocalStorage | REST API
